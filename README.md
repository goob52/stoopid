# Fourteen Walls

A 14×14 riichi mahjong falling-tile puzzle game.

## Run locally

Double-click `index.html`.

## Publish with GitHub Pages

1. Create a new repository on GitHub.
2. Upload every file from this folder.
3. Open the repository's **Settings**.
4. Open **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/ (root)`.
7. Save.

GitHub will provide a public link for the game.

## Controls

### Noob Mode
- Left / Right: move
- Up or X: rotate
- Down: move down one row
- Space: hard drop
- C: hold
- P: pause

### Pro Mode
- The piece falls automatically and speeds up after completed hands.
